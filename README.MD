**Информация для разработчиков античита:**

Все действия производятся за счет функции **SendPacket**.

Отправляет всякие разные пакеты, для фикса нужно лишь заблокировать использования функции из читов.

Например установить jmp (detour) на функцию SendPacket и проверять там _ReturnAddress() (Если вызов произошел не из Game.dll то блокировать)

Или заставить бота парсить подозрительные пакеты сразу в бан (но нужно учитывать что начало у пакета может быть не с первого байта! не только с первого!)
